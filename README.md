# Vollocare Claude Plugins

Vollocare team 的 Claude Code plugin marketplace。

## 安裝 Marketplace

```bash
claude plugins add-marketplace vollocare/claude-plugins
```

## 安裝 Plugin

```bash
claude plugins install Get-email-records
```

## 可用 Plugins

| Plugin | 說明 |
|--------|------|
| [Get-email-records](https://github.com/vollocare/Get-email-records) | 從 Supabase 查詢 email 記錄，支援時間、類型、關鍵字篩選 |
| [Get-message-records](https://github.com/vollocare/Get-message-records) | 從 Supabase 查詢 message 記錄，支援時間、群組、產品類型、發言人、關鍵字篩選 |
| [tts_cli](https://github.com/vollocare/tts_cli) | 本地文字轉語音 CLI，針對 macOS Apple Silicon 最佳化，支援繁體中文，可選 Qwen3-TTS（快速）或 Fish Speech（高品質）引擎 |
